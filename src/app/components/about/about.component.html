Basic information

This is a panel that keeps track of particular clients and how much money they owe to a business. It uses Angular, Bootstrap and Google's Firebase as the backend.

Technologies used in this project:

  • Angular
  • Bootstrap 4
  • Font Awesome 4
  • Firebase with Firestore (beta)
  • angularfire2
  • angular2-flash-messages
